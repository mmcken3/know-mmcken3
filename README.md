# know-mmcken3

Personal site for [Mitchell McKenzie](https://github.com/mmcken3) — engineering leader, AI/cloud architect, and fitness coach.

The site covers three areas: engineering leadership and software craft, architecture and AI work, and fitness coaching (CrossFit, Olympic lifting, Hyrox).

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Framer Motion** — scroll-driven fade animations
- **next-themes** — light/dark mode
- **lucide-react** — icons

## Pages

| Route | Description |
|---|---|
| `/` | Hero + disciplines overview |
| `/about` | Background and experience |
| `/projects` | Work and side projects |
| `/blog` | Writing |
| `/fitness` | Coaching and training |

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Other commands

```bash
npm run build   # production build
npm run start   # serve production build locally
npm run lint    # run ESLint
```

## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` to deploy.
